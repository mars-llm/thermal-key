Quick Start

git clone https://github.com/mars-llm/thermal-key.git
cd thermal-key

python3 thermal.py -H 192.168.1.100 status
python3 thermal.py -H 192.168.1.100 watch

Safety

Heat is real

Miners can overheat or be damaged with incorrect settings. Use conservative limits and monitor temps.

Network control

CGMiner on port 4028 is unauthenticated. Use VLANs or firewall rules to keep your network safe.

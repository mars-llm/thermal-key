Quick Start

Install once, then control any miner on your LAN.

git clone https://github.com/mars-llm/thermal-key.git
cd thermal-key

python3 thermal.py -H 192.168.1.100 status
python3 thermal.py -H 192.168.1.100 watch

Command map

Everything you need for daily control in a single script.

Status + Pools

thermal.py -H IP status thermal.py -H IP pools

Fan + Mode

thermal.py -H IP fan 80 thermal.py -H IP mode 1

Fleet control

thermal.py -H ip1,ip2,ip3 reboot thermal.py -H miners.txt status

Auth + Recovery

thermal.py -H IP auth PASS thermal.py -H IP getauth

Security posture

The CGMiner API on port 4028 has no authentication. Treat it like a powerful local admin interface.

Isolate miners

Use a dedicated VLAN or firewall rules to keep port 4028 private.

Use strong passwords

Protect the web UI and rotate credentials after changes.
